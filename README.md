# Fs-customized-bash

This repository collects ".bashrc" and ".bash_profile" files currently used by Fsu0413.

It is a personal utility project that will be updated at any time.
The configurations are based on .bashrc provided by Ubuntu and MSYS2, with personal customizations (especially PS1).

This repository uses the WTFPLv2 license.

## Installation
To install, run `./install.sh` or source the desired files manually.

## Structure
- `public/`: Public configuration files
- `private/`: Private files (not meant to be committed)
